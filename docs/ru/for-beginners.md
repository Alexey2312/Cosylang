# Введение

## Cosylang --- высокоуровневый, мультипарадигменный, статически типизированный язык программирования для разработки приложений, CLI-утилит и скриптов автоматизации.
### Демоверсия кода:
``` C
body calculator
{

  pub func calculate(a: int32, operation: string, b: int32) -> int32
  {
    switch operation
    {
      case "+"
      {
        return add(a, b)
      }
    
      case "-"
      {
        return subtract(a, b)
      }
    
      case "*"
      {
        return multiply(a, b)
      }
    
      case "/"
      {
        return divide(a, b)
      }
    }
  }

  func add(a: int32, b: int32) -> int32
  {
    return a + b
  }
  
  func subtract(a: int32, b: int32) -> int32
  {
    return a - b
  }
  
  func multiply(a: int32, b: int32) -> int32
  {
    return a * b
  }
  
  func divide(a: int32, b: int32) -> int32
  {
    return a / b
  }
}

func main() -> void
{
  println(calculator::calculate(10, "+", 5))
}
```

# Базовые конструкции

## Импортирование файлов:

В процессе разработки могут понадобиться внешние модули или библиотеки. Для включения их в исходный код можно использовать ключевое слово `import`:

``` Java
import <std>
```

Но, если необходимо использовать файл, не относящийся к внутренним директориям компилятора, то можно использовать такой синтаксис :

``` Java
import "path/to/file.cosy"
```


## Переменные и константы:

### Переменные служат для хранения данных, которые могут изменяться в процессе выполнения программы. Константы же предназначены для хранения неизменяемых значений.
Переменные объявляются с помощью ключевого слова `var`, после которого указывается название переменной, а далее с помощью оператора ':' тип пременной.
```Go
var someVar: int32
```
Однако, такая запись вызовет ошибку компиляции, так как переменная не инициализирована. Чтобы инициализировать переменную, необходимо присвоить ей значение,
используя оператор присваивания '=':
``` Go
var someVar: int32 = 43
```
С помощью этого оператора можно изменить значение переменной после её инициализации:
``` Go
var someVar: int32 = 43
someVar = 49
```

### Объявление констант схоже с объявлением переменных, но с использованием ключевого слова `const`:
``` Go
const PI: float32 = 3.14
```
Важно помнить, что константы не могут быть изменены после инициализации, это ошибочная запись:
```GO
const PI: float32 = 3.14
PI = 14.3 // ОШИБКА!!!
```

## Функции в программе:

Для разделения кода на модули, можно использовать функции. Функции объявляются с помощью ключевого слова `func`:

``` Java
func functionName()
{
    // код функции
}
```

Функции могут принимать параметры и возвращать значения. Параметры функции заключаются в круглые скобки после названия и могут быть использованы внутри тела функции, определённого с помощью фигурных скобок:

``` Java
func someFunction(a: int32, b: int32)
{
    // a и b могут использоваться здесь.
}
```

Для указания типа возвращаемого из функции значения, используется оператор `->`, после которого указывается тип:

``` Go
func add(a: int32, b: int32) -> int32
{
    return a + b // Возвращаем сумму a и b с помощью ключевого слова 'return'
}
```

## Точка входа в программу:

Для запуска программы необходимо объявить функцию `main`, она будет вызвана сразу после компиляции проекта:

``` Java
func main() -> void
{
    // код, который будет выполнен при запуске программы
}
```

## Условные конструкции:

### if-else-else if

``` C
if someCondition 
{
    // код, который выполняется, если условие истинно
} 
else if anotherCondition 
{
    // код, который выполняется, если первое условие ложно, а второе истинно
} 
else 
{
    // код, который выполняется, если оба условия ложны
}
```

### switch-case:

``` C
switch expression
{
    case value1
    {
        // код, который выполняется, если expression равен value1
    }
    case value2
    {
        // код, который выполняется, если expression равен value2
    }
    default
    {
        // код, который выполняется, если expression не равен ни одному из значений
    }
}
```
